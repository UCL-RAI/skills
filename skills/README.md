# Skills

Published skills will live in this directory.

Suggested layout:

```text
skills/
|-- research/
|   `-- literature-review/
|       `-- SKILL.md
|-- robotics/
|   `-- simulation-debugging/
|       `-- SKILL.md
`-- engineering/
    `-- code-review/
        `-- SKILL.md
```

Each published skill should have a matching entry in `../catalog.json`.
