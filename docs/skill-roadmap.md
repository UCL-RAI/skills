# Skill Roadmap

This roadmap keeps the first public release small enough to review while covering the full Robotics & AI research loop.

## Release 0.1

| Skill | Type | Purpose |
| --- | --- | --- |
| `rai-research-flow` | flow | Route research onboarding, survey, project ideation, and paper-writing tasks. |
| `rai-paper-flow` | flow | Route conference-paper writing and audit workflows for Robotics & AI venues. |
| `rai-coding-flow` | flow | Route daily Robotics & AI code work through design, debugging, testing, review, and reproducibility checks. |
| `paper-search-protocol` | atomic | Build reproducible search logs across papers, projects, datasets, and code. |
| `paper-reading-card` | atomic | Convert one paper into a structured, source-grounded reading card. |
| `evidence-matrix-builder` | atomic | Turn paper/project cards into comparison matrices and gap maps. |
| `research-idea-rubric` | reference | Score and refine project ideas for novelty, feasibility, baselines, and evaluation. |
| `venue-paper-outline` | reference | Plan papers for ICRA, RSS, CoRL, ICML, ICLR, and NeurIPS-style venues. |
| `citation-integrity-auditor` | atomic | Check whether citations are real and actually support each claim. |
| `scientific-figure-director` | tool | Plan, prompt, and audit scientific figures and diagrams. |
| `paper-red-team-review` | atomic | Perform harsh but evidence-grounded paper review across facts, logic, writing, and visuals. |
| `robotics-ai-coding-flow` | atomic | Apply Robotics & AI code quality checks around ROS, simulation, ML experiments, and reproducibility. |

## Release 0.2 Candidates

- `knowledge-base-ingestion`: deterministic folder-to-index ingestion protocol for papers, notes, code, and datasets.
- `benchmark-audit`: validate benchmark choice, metrics, baselines, ablations, and statistical reporting.
- `experiment-provenance-auditor`: check seeds, configs, logs, environment capture, and result traceability.
- `reviewer-response-builder`: build point-by-point conference reviewer responses.
- `slide-talk-builder`: generate lab meeting, journal club, and conference talk decks from papers or project notes.

## Non-Goals

- Do not create a single mega-skill that claims to do all research.
- Do not bundle third-party repositories.
- Do not encode venue deadlines or time-sensitive rules in skill bodies.
- Do not make paper-writing skills fabricate citations, benchmarks, or claims.
