# EARL Brand Assets

This repository uses EARL Lab as its primary visual and ownership identity. UCL affiliation is expressed in text; no UCL logo is used in repository or organisation visuals.

## Approved Assets

| Asset | Use |
| --- | --- |
| [`masthead-dark.svg`](./assets/earl/masthead-dark.svg) | README and profile masthead on dark themes. |
| [`masthead-light.svg`](./assets/earl/masthead-light.svg) | README and profile masthead on light themes. |
| [`avatar-square-dark-512.png`](./assets/earl/avatar-square-dark-512.png) | GitHub organisation avatar and other square profile placements. |

The full local logo source package is not part of this repository. Only assets required by public repository surfaces are versioned here.

## Identity Hierarchy

Use this order consistently:

1. Primary identity: **EARL Lab**.
2. Full name: **Embodied Autonomy and Robot Learning Lab**.
3. Affiliation: **UCL Robotics & AI at University College London**.

Approved long-form affiliation:

> EARL Lab is an Embodied Autonomy and Robot Learning research group within UCL Robotics & AI at University College London.

Approved short-form attribution:

> Developed and maintained by EARL Lab within UCL Robotics & AI.
