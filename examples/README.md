# Examples

These examples show realistic use patterns for the highest-value draft skills.

They are not benchmark results, literature claims, or validated research outputs. They show the expected artifact shape and the level of evidence discipline a skill should enforce.

Each example uses placeholders such as `<verified source URL>` when a real-time search or paper read is required. Do not replace those placeholders with invented citations.
